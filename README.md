# Schoolab

**Schoolab** est la solution complète pour la gestion efficace de votre établissement scolaire. Elle vous permet de gérer les inscriptions, les classes, les notes et de générer automatiquement les bulletins scolaires, évitant ainsi les erreurs de calcul manuelles.

## Fonctionnalités Principales

- **Gestion des Élèves** : Inscriptions, dossiers complets, suivi.
- **Gestion des Classes** : Organisation pédagogique structurée.
- **Calcul des Notes** : Calcul automatique des moyennes et classements.
- **Bulletins Scolaires** : Génération et impression de bulletins professionnels.
- **Synchronisation** : Sauvegarde des données en locale et synchronisation cloud possible.

## Installation

Téléchargez le paquet `.deb` pour votre distribution Linux et installez-le via votre gestionnaire de paquets habituel (ex: `sudo dpkg -i schoolab_1.0.9_amd64.deb`).
